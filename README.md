# DataForge Engine v1.0

Single-file data processing pipeline built with **pure JavaScript Standard Library**  
Final exam project — JavaScript Standard Library module

## Features Demonstrated

- Number formatting & parsing
- String normalization, slugification, truncation
- Array queue/stack, search, filter, transform
- Object merging, freezing, proxy validation
- JSON import/export
- BigInt large inventory calculation
- Date timestamps & formatting
- Math statistics (min/max/avg)
- Boolean flag wrapper
- Map & Set collections
- Symbol metadata
- RegExp validation & extraction
- URI & Base64 encoding
- Dynamic formula evaluation (safe `new Function`)

## Requirements

- Node.js 18+
- No external dependencies

## Usage

```bash
node dataforge.js
